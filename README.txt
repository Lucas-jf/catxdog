Atividade avaliativa - Práticas com javascript

Instruções

    Crie um repositório no github
    Faça commits atualizando seu trabalho
    Escolha uma API anexa, descreva-a no README e desenvolva uma solução usando-a (DICA: Use fetch)
    Estilize seu site
    Anexe o link do seu repositório e entregue a atividade

Critérios de avaliação:

    O código executado não exibe erros no console
    O código foi versionado com mensagens significativas
    O código implementa corretamente requisições à API escolhida
